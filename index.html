📘 Manual do Usuário

🧮 Calculadora
• Display de expressão: toque para selecionar/copiar. Edição exclusiva pelos botões (sem teclado do sistema).
• Use ← → virtuais para mover o cursor. DEL apaga caractere, CLR limpa tudo.
• Teclas expandidas: STO, RCL, ENG, CONST, CONV, Pol, Rec, Ran#, PreAns, GCD, LCM.
• Modo Científico, Conversor, Programador, Gráfico, Derivada/Integral numérica.

🎮 Jogo
• START: escolha avatar. Setas ou teclado para mover, espaço/toque para pular.
• Plataformas: +2 pontos. Inimigos: aparecem após 25s (+5 pts ao pisar). Escudos: após 30s (proteção, sem pontos).
